# Bingo Game SImulator

- To run download the BingoGameSim.jar file
- Either just click to run
- Or you can run from the command line using: java -jar BingoGameSim.jar

# How it works

- On launch you will get assigned 16 numbers as well as two other rival players who each have a random 16 numbers selected
- Simply click the button to generate a new number
- If it is in our list it will remove it, as it will for your rivals list
- Keep drawing balls until there is a winner
- You can reset the game and keep playing
- It will keep a tally of your win rate at the bottom
